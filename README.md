>1.首先利用brupsuite软件抓包分析一了一下请求结构。
>2.利用python那~~真正给人使用的~~强大的requests库模拟一个请求并提交并得到服务器的应答消息
>3.利用一个将python程序转换成exe文件的小工具保证一定的移植性
